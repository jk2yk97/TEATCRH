La malla curricular del Técnico Especialista en Atención al Cliente y Recepción Hotelera (TEACRH) es una guía visual para organizar tu avance en la formación hotelera. Te permite marcar fácilmente las materias que ya completaste y ver de forma clara tu progreso por semestres, ayudándote a llevar un seguimiento ordenado y motivador de tu capacitación profesional.
